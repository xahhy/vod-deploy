# Deploy Use Python Fabric
## Install
1. create virtualenv 

```shell
virtualenv python2.7 env
```

2. activate virtualenv

```shell
. env/bin/activate
```
> use `deactivate` to leave virtualenv

3. install fabric use pip

```shell
pip install fabric
```
## Usage
- list all available task use `fab -l`
- run one of the task use `fab <task name>`
